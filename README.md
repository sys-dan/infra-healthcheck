ㅅ# infra-healthcheck

리눅스 서버(RHEL)를 대상으로 서버 상태를 자동 점검하는 스크립트
CPU, 메모리, 디스크, 네트워크, 서비스 상태 및 로그 정보를 수집하여
장애 발생 시 문제 가능성이 높은 영역을 신속하게 식별하는 것을 목표

## 목적

본 내용은 시스템 대상으로 한 표준화된 점검 스크립트를 제공함으로써,
장애 발생 시 원인 파악까지의 시간을 단축하고 인프라 운영 효율성을 향상시키는 것을 목표
CPU, 메모리, 디스크, 네트워크, 주요 서비스 상태 및 로그 정보를 자동으로 수집·분석하여,
장애 가능성이 높은 구간을 신속하게 식별할 수 있도록 지원함으로써 MTTR(Mean Time To Recovery)을 감소시키고 서비스 가용성과 운영 품질을 지속적으로 개선
